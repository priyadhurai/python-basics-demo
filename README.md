# Python Basics Demonstration

This repository contains a clean and complete Python script that demonstrates essential foundational concepts.  
It is designed as a beginner-friendly project to showcase Python skills for job applications, learning, and practice.

---

## Contents

The project includes examples of:

### ✓ Variables & Data Types  
- Strings  
- Integers  
- Floats  
- Booleans  

### ✓ Data Structures  
- List  
- Tuple  
- Set  
- Dictionary  

### ✓ Control Flow  
- If–else conditions  
- For loops  
- While loops  

### ✓ Functions  
- User-defined functions  
- Parameters and return values  
- Simple business logic (bonus calculation)

### ✓ File Handling  
- Writing to files  
- Reading from files using `with open()`

### ✓ Exception Handling  
- Try–except block  
- Handling errors gracefully

### ✓ Simple Data Processing  
- Aggregation  
- Calculating totals and averages
